# Castaño Logistics

Aplicación web para gestión de rutas de supervisores de terreno.

## Funcionalidades

- **Zonales**: Gestión de rutas de su equipo de supervisores
- **Supervisores**: Ver rutas planificadas y rendir gastos

## Tecnologías

- Streamlit
- Google Cloud Spanner (Graph)
- Google BigQuery

## Ejecutar Localmente

```bash
pip install -r requirements.txt
streamlit run app_logistics.py
```
